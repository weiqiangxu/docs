# 栈和队列

## 一、栈

### 1. 定义
栈是一种只能从表的一端（栈顶）存取数据的线性存储结构，遵循**后进先出**（Last In First Out, LIFO）原则。

### 2. 基本概念
- **栈顶**：允许数据插入和删除的一端
- **栈底**：固定的，不允许数据插入和删除的一端
- **压栈**：向栈中插入元素的操作
- **弹栈**：从栈中删除元素的操作

### 3. 存储结构
- **顺序栈**：使用数组实现，便于随机访问
- **链式栈**：使用链表实现，便于动态扩展

### 4. 时间复杂度
- 压栈：O(1)
- 弹栈：O(1)
- 查看栈顶元素：O(1)

### 5. 应用场景
- **浏览器前进后退**：浏览历史记录管理
- **函数调用**：调用栈管理
- **表达式求值**：中缀表达式转后缀表达式
- **括号匹配**：检查代码中的括号是否匹配
- **进制转换**：将十进制数转换为其他进制

## 二、队列

### 1. 定义
队列是一种只能从一端（队尾）插入数据，从另一端（队头）删除数据的线性存储结构，遵循**先进先出**（First In First Out, FIFO）原则。

### 2. 基本概念
- **队头**：允许删除元素的一端
- **队尾**：允许插入元素的一端
- **入队**：向队列中插入元素的操作
- **出队**：从队列中删除元素的操作

### 3. 存储结构
- **顺序队列**：使用数组实现，需要处理假溢出问题
- **链式队列**：使用链表实现，便于动态扩展
- **循环队列**：使用数组实现，解决假溢出问题

### 4. 时间复杂度
- 入队：O(1)
- 出队：O(1)
- 查看队头元素：O(1)

### 5. 应用场景
- **任务调度**：操作系统中的进程调度
- **消息队列**：异步消息处理
- **缓冲区**：IO操作的缓冲区
- **广度优先搜索**：图的遍历算法
- **打印机队列**：打印任务管理

## 三、栈与队列的区别

| 特性 | 栈 | 队列 |
|------|------|------|
| 操作端 | 只在栈顶操作 | 在队头和队尾分别操作 |
| 原则 | 后进先出（LIFO） | 先进先出（FIFO） |
| 应用场景 | 函数调用、括号匹配 | 任务调度、消息队列 |
| 实现方式 | 顺序栈、链式栈 | 顺序队列、链式队列、循环队列 |

## 四、特殊队列

### 1. 双端队列
- **定义**：允许在两端进行插入和删除操作的队列
- **应用**：滑动窗口算法、LRU缓存

### 2. 优先队列
- **定义**：元素具有优先级，出队时按优先级顺序出队
- **实现**：通常使用堆实现
- **应用**：任务调度、事件处理

### 3. 阻塞队列
- **定义**：当队列为空时，出队操作会阻塞；当队列满时，入队操作会阻塞
- **应用**：生产者-消费者模式

## 五、代码示例

### 1. 栈的实现（Python）
```python
class Stack:
    def __init__(self):
        self.items = []
    
    def push(self, item):
        self.items.append(item)
    
    def pop(self):
        if not self.is_empty():
            return self.items.pop()
        return None
    
    def peek(self):
        if not self.is_empty():
            return self.items[-1]
        return None
    
    def is_empty(self):
        return len(self.items) == 0
    
    def size(self):
        return len(self.items)
```

### 2. 队列的实现（Python）
```python
class Queue:
    def __init__(self):
        self.items = []
    
    def enqueue(self, item):
        self.items.append(item)
    
    def dequeue(self):
        if not self.is_empty():
            return self.items.pop(0)
        return None
    
    def front(self):
        if not self.is_empty():
            return self.items[0]
        return None
    
    def is_empty(self):
        return len(self.items) == 0
    
    def size(self):
        return len(self.items)
```
