## 三、常见操作命令

### 3.1 Debian 系命令

```bash
# 更新软件包列表
sudo apt update

# 升级所有可升级的包
sudo apt upgrade

# 安装软件包
sudo apt install <package-name>

# 卸载软件包
sudo apt remove <package-name>

# 搜索软件包
apt search <keyword>

# 显示软件包信息
apt show <package-name>

# 清理不再需要的依赖
sudo apt autoremove
```

### 3.2 RHEL 系命令

```bash
# 检查更新
sudo yum check-update

# 安装软件包
sudo yum install <package-name>

# 更新软件包
sudo yum update <package-name>

# 卸载软件包
sudo yum remove <package-name>

# 搜索软件包
yum search <keyword>

# 列出已安装的包
yum list installed

# 清理缓存
yum clean all
```
