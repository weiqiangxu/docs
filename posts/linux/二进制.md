## 二、有符号整数范围

### 2.1 int8/int32 取值范围

```mermaid
flowchart TB
    subgraph int8 8位
        A1[0 1111111] --> B1[+127 最大值]
        A2[1 0000000] --> B2[-128 最小值]
    end
    
    subgraph int32 32位
        A3[0 31个1] --> B3[+2147483647 最大值<br/>2³¹-1]
        A4[1 31个0] --> B4[-2147483648 最小值<br/>-2³¹]
    end
```

| 类型 | 范围 | 公式 |
|------|------|------|
| **int8** | -128 ~ +127 | -2⁷ ~ 2⁷-1 |
| **int32** | -2147483648 ~ +2147483647 | -2³¹ ~ 2³¹-1 |
| **uint8** | 0 ~ 255 | 0 ~ 2⁸-1 |

## 四、Golang 移位比较大小

```go
// 通过符号位比较两个整数大小
func compare(a, b int) string {
    c := a - b
    z := (unsafe.Sizeof(c)-1)*8 + 7  // 计算符号位位置
    d := c >> z                      // 算术右移
    l := []string{"b大", "a大"}
    return l[d+1]
}
```

```mermaid
flowchart LR
    A[a - b] --> B[计算符号位]
    B --> C[算术右移]
    C --> D{结果}
    D -->|< 0|E[b大]
    D -->|> 0|F[a大]
```
