int i,sum =0,n=100

 ```

```

### 算法时间复杂度定义

总执行次数 T ,分析 T 和 n 的关系，算出 T 的数量级, 算法的时间复杂度也就是算法的时间度量；

T(n)  = O(f(n))

f(n) 是算法之中的某一个函数

个人理解:

次数 T

执行动作 f()

执行条件 n

复杂度 O

### 常见的时间复杂度级别

```
O(1)  常数

O(n)  线性

O(n^2)  平方
```

### 空间复杂度

定义

```
算法的空间复杂度通过计算算法所需的存储空间实现，算法空间复杂度的计算公式记作：S(n)=O(f(n))，其中，n为问题的规模，f(n)为语句关于n所占存储空间的函数
```
