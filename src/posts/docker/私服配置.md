### 1.进入镜像

```bash
mkdir -p /root/.docker && touch /root/.docker/config.json
mkdir -p /etc/docker && touch /etc/docker/daemon.json
```

### 2.config.json

```json
{
  "auths": {
    "registry.my.net": {
      "auth": "xxx"
    }
  }
}
```

### 3.daemon.json

```json
{
  "insecure-registries" : [ "registry.my.net" ]
}
```
