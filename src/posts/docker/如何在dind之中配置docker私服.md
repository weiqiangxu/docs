### 进入镜像

```
mkdir -p /root/.docker && touch /root/.docker/config.json
mkdir -p /etc/docker && touch /etc/docker/daemon.json
```

### config.json
```
{
  "auths": {
    "registry.bingosoft.net": {
      "auth": "xxx"
    }
  }
}
```

### daemon.json
```
{
  "insecure-registries" : [ "registry.bingosoft.net" ]
}
```
