``` bash
# ip link add <interface_name> type <interface_type>
$ ip link add eth0 type vlan

```

### 在`<interface_type>`中，您可以使用以下选项来指定不同的网卡类型：

- `ethernet`：以太网接口
- `bridge`：桥接接口
- `vlan`：虚拟局域网接口
- `tun`：TUN/TAP接口
- `wifi`：无线接口
- `bond`：绑定接口
- `veth`：虚拟以太网接口
- `macvlan`：MACVLAN接口
