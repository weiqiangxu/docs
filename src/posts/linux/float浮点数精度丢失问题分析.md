> 整型是用的二进制存储数据，保留一个符号位(unsigned就没有)

> 浮点数则用的是科学计数法存储数据

### float同样也使用4字节存储，但除了整数外还能够存储小数位，并且表示的范围比int还要更大，怎么做到的？

```
与int的存储方式不同，float为了支持小数，因此采用类似科学记数法进行数据的表示
```

```
1 = 10^0
0.00001= 1 * 10^-5
610 = 6.1 * 10^2
0.13 = 1 * 10^-1 + 3 * 10^-2
```
