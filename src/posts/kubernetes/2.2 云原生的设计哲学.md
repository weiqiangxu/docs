# 云原生的设计哲学


### 设计理念

```
分布式设计（Distribution）：容器、微服务、API 驱动的开发；
配置设计（Configuration）：一个镜像，多个环境配置；
韧性设计（Resistancy）：故障容忍和自愈；
弹性设计（Elasticity）：弹性扩展和对环境变化（负载）做出响应；
交付设计（Delivery）：自动拉起，缩短交付时间；
性能设计（Performance）：响应式，并发和资源高效利用；
自动化设计（Automation）：自动化的 DevOps；
诊断性设计（Diagnosability）：集群级别的日志、metric 和追踪；
安全性设计（Security）：安全端点、API Gateway、端到端加密；
```
